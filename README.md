# ✨ Multi-Step Form Design and Validation

A premium multi-step registration form built using HTML, CSS, and JavaScript. This project provides a modern and interactive user experience with real-time form validation, animated progress indicators, and a responsive glassmorphism design.

---

## 🚀 Features

- 📝 Multi-Step Registration Form
- ✅ Real-Time Form Validation
- 📊 Animated Progress Bar
- 👤 Profile Image Upload & Preview
- 🔒 Password Strength Indicator
- 👁️ Show/Hide Password Feature
- 🌙 Dark Mode Toggle
- 📱 Fully Responsive Design
- 🎨 Modern Glassmorphism UI
- 📍 Step-by-Step Navigation
- 💾 Local Storage Support
- 🎉 Registration Success Message

---

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)
